# OctoAcme Project Management Documentation

Welcome to the OctoAcme Project Management Docs repository. This documentation provides a comprehensive guide to the project management processes, workflows, and best practices used at OctoAcme.

## About OctoAcme Project Management

OctoAcme follows a structured, customer-first approach to project delivery with the following core principles:

- **Customer-first**: Prioritize customer value and usability in every decision
- **Iterative delivery**: Deliver small, testable increments and gather feedback early
- **Clear ownership**: Each project has named roles with clear accountability (Project Manager, Product Manager, Development Team)
- **Data-informed decisions**: Measure impact and iterate based on evidence
- **Psychological safety**: Encourage feedback, learning, and continuous improvement

Our project lifecycle spans five key phases: Initiation, Planning, Execution, Release, and Retrospective. Each phase has defined activities, deliverables, and decision gates to ensure successful project delivery.

## How to Use This Documentation

- **New team members**: Start with the [Project Management Overview](octoacme-project-management-overview.md) to understand the framework, then read the phase-specific guides as you work on projects.
- **Active project teams**: Keep your project charter in sync with your repo and reference the relevant phase guides during execution.
- **Process improvements**: Use the [Retrospective & Continuous Improvement](octoacme-retrospective-and-continuous-improvement.md) guide to capture feedback and suggest updates to these docs.

## Key Artifacts Used Across Projects

- **Project Charter / One-pager**: Defines problem, goals, success metrics, and key stakeholders
- **Risk Register**: Tracks identified risks, impacts, mitigation plans, and status
- **Sprint/Iteration Backlog**: Prioritized list of work items with acceptance criteria
- **Release Plan & Roadmap**: Milestone timeline and release strategy
- **Retrospective Notes**: Lessons learned and action items for continuous improvement
